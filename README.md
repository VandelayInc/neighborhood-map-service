### Usage

The HTML page is found at `/rooms/:roomid`

The API endpoint is `/api/:roomid`

### Installation

Make sure MongoDB is started. From within the root directory:

```sh
$ npm install
$ node seed.js
$ npm run react-dev
$ npm start
```
